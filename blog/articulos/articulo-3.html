Guía rápida: Deploy con Netlify

20 de Octubre, 2025

Si vienes del mundo del hosting tradicional (subir archivos por FTP, gestionar servidores), el despliegue (deploy) de sitios estáticos modernos puede parecer magia. Netlify es una de las plataformas líderes que ha simplificado este proceso al extremo, integrando el despliegue continuo (CI/CD) directamente con tu repositorio de Git.

En esta guía, desplegaremos un sitio estático en minutos.

Paso 1: Prerrequisitos

Un sitio estático: Puede ser HTML/CSS/JS simple, o un proyecto construido con un generador como React (Create React App), Vue, Astro o Svelte.

Un repositorio en Git: Tu código debe estar alojado en GitHub, GitLab o Bitbucket.

Una cuenta en Netlify: Registrarse es gratis.

Paso 2: Conectar tu Repositorio

Una vez dentro de tu panel de Netlify: haz clic en "Add new site" → "Import an existing project". Conecta tu proveedor de Git (ej. GitHub) y autoriza a Netlify. Selecciona el repositorio que quieres desplegar.

Paso 3: Configurar el Build

Aquí es donde Netlify necesita saber cómo "construir" tu sitio (si no es HTML simple).

  • Branch to deploy: Generalmente main o master.
  • Build command: Si usas React o Vue, aquí pondrías npm run build. Si es HTML simple, puedes dejarlo vacío.
  • Publish directory: Es la carpeta que genera el comando anterior. En React (CRA) es build, en Vite es dist. Si es HTML simple, es el directorio raíz (.).

Paso 4: ¡Deploy!

Haz clic en "Deploy site". Netlify clonará tu repositorio, ejecutará el comando de construcción (si lo hay) y desplegará los archivos del directorio de publicación en su CDN global. En menos de un minuto, te dará una URL aleatoria (ej. tu-sitio-genial.netlify.app) donde tu sitio ya está en línea.

Paso 5 (Extra): La Magia del CI/CD

Ahora viene lo mejor. Realiza un cambio en tu código local, haz git commit y luego git push a tu rama main. Netlify detectará automáticamente ese push, volverá a ejecutar el proceso de build y actualizará tu sitio en vivo. No necesitas hacer nada más.

Conclusión

Netlify elimina toda la fricción del despliegue. Con funciones adicionales como deploy previews (una vista previa por cada pull request), manejo de formularios y HTTPS automático, se ha convertido en la herramienta de referencia para el desarrollo web moderno estático.
