Guía rápida: Deploy con Netlify

20 de Octubre, 2025

Si vienes del mundo del hosting tradicional (subir archivos por FTP, gestionar servidores), el despliegue (deploy) de sitios estáticos modernos puede parecer magia. Netlify es una de las plataformas líderes que ha simplificado este proceso al extremo, integrando el despliegue continuo (CI/CD) directamente con tu repositorio de Git.

En esta guía, desplegaremos un sitio estático en minutos.

Paso 1: Prerrequisitos

Antes de empezar, asegúrate de tener lo siguiente:

  • Un sitio estático: HTML/CSS/JS simple o un proyecto React/Vue.
  • Un repositorio en Git: Tu código subido a GitHub, GitLab o Bitbucket.
  • Una cuenta en Netlify: Registrarse es gratis.

Paso 2: Conectar tu Repositorio

Una vez dentro de tu panel de Netlify: haz clic en "Add new site" → "Import an existing project". Conecta tu proveedor de Git (ej. GitHub) y autoriza a Netlify. Selecciona el repositorio que quieres desplegar.

Paso 3: Configurar el Build

Aquí es donde Netlify necesita saber cómo "construir" tu sitio (si no es HTML simple).

  • Branch to deploy: Generalmente main o master.
  • Build command: Si usas React o Vue, aquí pondrías npm run build. Si es HTML simple, puedes dejarlo vacío.
  • Publish directory: La carpeta que genera el comando. (ej. build o dist).

Paso 4: ¡Deploy!

Haz clic en "Deploy site". Netlify clonará tu repositorio, ejecutará el comando de construcción y desplegará los archivos en su CDN global. En menos de un minuto, te dará una URL aleatoria donde tu sitio ya está en línea.

Paso 5 (Extra): La Magia del CI/CD

Ahora viene lo mejor. Realiza un cambio en tu código local, haz git commit y luego git push. Netlify detectará automáticamente ese push y actualizará tu sitio en vivo.

"CI/CD (Integración y Despliegue Continuos) significa que ya no necesitas subir archivos manualmente. Tu repositorio es la única fuente de verdad."

Conclusión

Netlify elimina toda la fricción del despliegue. Con funciones adicionales como deploy previews, manejo de formularios y HTTPS automático, se ha convertido en la herramienta de referencia para el desarrollo web moderno estático.
