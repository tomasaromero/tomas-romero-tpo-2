Crear tu primer componente en React

               

20 de Octubre, 2025

               

React revolucionó el desarrollo web al popularizar la "programación basada en componentes". La idea es simple: en lugar de pensar en páginas enteras, construyes piezas de UI pequeñas, aisladas y reutilizables llamadas componentes.

               

Vamos a construir el componente más común de todos: un botón, pero haciéndolo interactivo.

               

Paso 1: Configuración

               

Asumimos que ya tienes un proyecto de React funcionando (usando npx create-react-app o npm create vite@latest).

               

Paso 2: El Componente Funcional

               

En React moderno, los componentes son simplemente funciones de JavaScript que devuelven JSX (una sintaxis similar a HTML).

               

Crea un nuevo archivo: src/components/Contador.js

                
// src/components/Contador.js
import React from 'react';

// Este es un componente funcional.
// Su nombre debe empezar con mayúscula.
function Contador() {
  // Lo que retornamos es JSX, que describe la UI.
  return (
    <div>
      <p>Has hecho clic 0 veces</p>
      <button>
        Hazme clic
      </button>
    </div>
  );
}

export default Contador;
               

Paso 3: Usar el Componente

               

Ahora, en tu archivo principal src/App.js, puedes importar y usar tu componente como si fuera una etiqueta HTML:

                
// src/App.js
import React from 'react';
import Contador from './components/Contador'; // Importamos nuestro componente

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <h1>Mi Primer Componente</h1>
        <Contador />
      </header>
    </div>
  );
}

export default App;
               

Paso 4: Introduciendo el Estado (Hooks)

               

El "estado" (state) es la memoria de un componente. Para añadir estado a un componente funcional, usamos un Hook llamado useState.

                
// src/components/Contador.js
import React, { useState } from 'react';

function Contador() {
  const [conteo, setConteo] = useState(0);

  const manejarClic = () => {
    setConteo(conteo + 1);
  };

  return (
    <div>
      <p>Has hecho clic {conteo} veces</p>
      <button onClick={manejarClic}>
        Hazme clic
      </button>
    </div>
  );
}

export default Contador;
               

Conclusión

               

¡Felicidades! Acabas de crear un componente de React con estado. React ahora "observará" la variable conteo. Cada vez que llames a setConteo(), React actualizará el valor y automáticamente volverá a renderizar el componente (y solo ese componente) para mostrar el nuevo valor en la pantalla. Esta es la magia de la UI declarativa.
